# Welcome to Flint

Flint is a structured workspace system designed for knowledge work. It provides a consistent framework for organizing documents, tracking work, and collaborating with AI agents.

## What is Flint?

Flint combines:

- **Mesh** — A structured folder for your working documents
- **Plugins** — Extensible capabilities and workflows
- **Workspaces** — References to external codebases and resources
- **Exports** — Polished outputs for sharing
- **Connections** — Inter-flint communication
